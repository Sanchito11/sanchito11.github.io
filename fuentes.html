Las mejores fuentes web seguras para HTML y CSS

La siguiente lista son las mejores fuentes web seguras para HTML y CSS:

- Arial (sans-serif)
- Verdana (sans-serif)
- Tahoma (sans serif)
- Trebuchet MS (sans serif)
- Times New Roman (serif)
- Georgia (serif)
- Garamond (serif)
- Courier Nuevo (monoespaciado)
- Brush Script MT (cursiva)

Como utilizar fuentes de Google

Primeramente entrariamos en la siguiente página donde eligiremos el tipo de fuente que vayamos a usar:

Google Fonts

Una vez elegida, la seleccionamos y copiariamos la url correspondiente para ponerla en el <header> de nuestro html.


Después crearemos un CSS donde podamos llamar a la fuente para introducirla, por un CSS externo o introduciendolo en linea como ya sabemos:

Ejemplo: <p font-family: 'Fuzzy Bubbles', cursive;>
